Create a one-dimensional histogram.

Parameters:

  • x (pandas.Series) ‚Äì Data from which to create bins.

  • y (pandas.Series or None, optional) ‚Äì Values to aggregate in bins of x. If None, counts of x are used.

  • logx (bool, optional) ‚Äì If True, compute bins in logarithmic space.

  • axnorm ({"t", "d", None}, optional) ‚Äì Normalisation applied to the histogram. "t" uses total counts and "d" yields a density.

  • clip_data (bool, optional) ‚Äì Remove extreme values at the 0.001 and 0.999 percentiles before binning or aggregation.

  • nbins (int or array-like, optional) ‚Äì Binning strategy passed to numpy.histogram_bin_edges() or pandas.cut() depending on the input type.

  • bin_precision (int, optional) ‚Äì Precision for decimal bin edges.

Parameters:

  • ax ‚Äì Axis on which to plot.

  • smooth (bool) ‚Äì If True, apply a Savitzky-Golay filter (scipy.signal.savgol_filter()) to the y-values before plotting to smooth the curve.

  • sg_kwargs (dict, None) ‚Äì If not None, dict of kwargs passed to Savitzky-Golay filter. Also allows for setting of window_length and polyorder as kwargs.
