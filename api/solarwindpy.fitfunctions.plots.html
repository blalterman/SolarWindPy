plot_residuals(ax=None, pct=True, subplots_kwargs=None, kind='both', **kwargs)[source]

Plot residuals for the data used in the fit.

Residuals are shown on a symlog scale with linthresh=10. By default they are expressed as percentages of the fitted model.

Parameters:

  • ax (mpl.axes.Axes, optional) ‚Äì Axis to draw on. If None a new figure and axis are created.

  • pct (bool, default True) ‚Äì If True, show residuals as percentages.

  • subplots_kwargs (dict, optional) ‚Äì Passed to plt.subplots when ax is None.

  • kind ({'simple', 'robust', 'both'}, default "both") ‚Äì Which residuals to plot.

  • **kwargs ‚Äì Additional keyword arguments passed to ax.plot.

Returns:

ax ‚Äì Axis with residuals plotted.
