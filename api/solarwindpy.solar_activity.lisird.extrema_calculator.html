Tools for calculating extrema in LISIRD activity indices.

class ExtremaCalculator(name, activity_index, threshold=None, window=600)[source]ÔÉÅ

Bases: object

Determine extrema in an activity index time series.

The calculator smooths the input series with a rolling mean and finds local minima and maxima based on a threshold value.
