Parameters:

  • x (pd.Series) – Data defining (x, y) coordinates.

  • z (pd.Series, optional) – If not None, used to specify the color for each point.

  • clip_data (bool) – If True, remove extreme values at the 0.001 and 0.999 percentitles.
