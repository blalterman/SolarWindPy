calculate_extrema_kind()[source]

Determine whether an SSN observation belongs to a cycle Minimum or Maximum.

Min: ssn < cycle mean Max: ssn >= cycle mean

Labels for the extremum kind include indicators for which cycle the extremum belongs to. These can be easily stripped with string manipulations, but are difficult to calculate after the fact.
