class HingeSaturation(xobs, yobs, guess_xh: float = 326, guess_yh: float = 0.5, **kwargs)[source]

Bases: FitFunction

Piecewise linear function with hinge point for saturation modeling.

The model consists of two linear segments joined at a hinge point (xh, yh):

  • Rising region (x < xh): \(f(x) = m_1 (x - x_1)\)

  • Plateau region (x >= xh): \(f(x) = m_2 (x - x_2)\)

where the slopes and intercepts are related by continuity at the hinge:

  • \(m_1 = y_h / (x_h - x_1)\)

  • \(x_2 = x_h - y_h / m_2\)

xh

Hinge x-coordinate (fitted parameter).

Type:

float

yh

Hinge y-coordinate (fitted parameter).

Type:

float

x1

x-intercept of rising line (fitted parameter).

Type:

float

m2

Slope of plateau region (fitted parameter). m2=0 gives constant saturation.

Type:

float

Examples

>>> import numpy as np
>>> from solarwindpy.fitfunctions import HingeSaturation
>>> x = np.linspace(0, 15, 100)
>>> y = np.where(x < 5, 2*x, 10)  # Saturation at y=10 for x>=5
>>> fit = HingeSaturation(x, y, guess_xh=5, guess_yh=10)
>>> fit.make_fit()
>>> print(f"Hinge at ({fit.popt['xh']:.2f}, {fit.popt['yh']:.2f})")
Hinge at (5.00, 10.00)

class TwoLine(xobs, yobs, guess_xs: float = 425.0, **kwargs)[source]

Bases: FitFunction

Piecewise linear function with two intersecting lines using minimum.

The model consists of two linear segments:

\[f(x) = \min(y_1, y_2)\]

where:

  • \(y_1 = m_1 (x - x_1)\)

  • \(y_2 = m_2 (x - x_2)\)

The lines intersect at the saturation point \((x_s, s)\) where:

  • \(x_s = \frac{m_1 x_1 - m_2 x_2}{m_1 - m_2}\)

  • \(s = m_1 (x_s - x_1) = m_2 (x_s - x_2)\)

x1

x-intercept of first line (fitted parameter).

float

x2

x-intercept of second line (fitted parameter).

Type:

float

Type:

Type:

float

xs

x-coordinate of intersection point (derived property).

Type:

float

s

y-coordinate of intersection point (derived property).

Type:

float

theta

Angle between the two lines in radians (derived property).

Type:

float

Examples

>>> import numpy as np
>>> from solarwindpy.fitfunctions import TwoLine
>>> x = np.linspace(0, 15, 100)
>>> y = np.minimum(2*(x-0), -1*(x-15))  # Two lines intersecting at (5, 10)
>>> fit = TwoLine(x, y, guess_xs=5.0)
>>> fit.make_fit()
>>> print(f"Intersection at ({fit.xs:.2f}, {fit.s:.2f})")
Intersection at (5.00, 10.00)

class Saturation(xobs, yobs, guess_xs: float = 425.0, guess_s: float = 0.5, **kwargs)[source]

Bases: FitFunction

Piecewise linear function reparameterized for saturation analysis.

This is an alternative parameterization of TwoLine where the saturation point coordinates and the angle between lines are used directly as parameters:

\[f(x) = \min(y_1, y_2)\]

Parameters are \((x_1, x_s, s, \theta)\) where:

  • \(x_1\): x-intercept of the rising line

  • \(x_s\): x-coordinate of saturation point

  • \(s\): y-coordinate of saturation point

  • \(\theta\): angle between the two lines (radians)

The slopes are derived as:

  • \(m_1 = s / (x_s - x_1)\)

  • \(m_2 = \tan(\arctan(m_1) - \theta)\)

class HingeMin(xobs, yobs, guess_h: float = 400.0, **kwargs)[source]

Bases: FitFunction

Piecewise linear function with hinge point using minimum.

The model consists of two linear segments joined at a hinge point:

\[f(x) = \min(y_1, y_2)\]

where:

  • \(y_1 = m_1 (x - x_1)\)

  • \(y_2 = m_2 (x - x_2)\)

Both lines pass through the hinge point \((h, y_h)\) where \(y_h = m_1 (h - x_1)\). The second slope is constrained by:

\[m_2 = m_1 \frac{h - x_1}{h - x_2}\]

Type:

float

Slope of second line (derived property).

Type:

float

class HingeMax(xobs, yobs, guess_h: float = 400.0, **kwargs)[source]

Bases: FitFunction

Piecewise linear function with hinge point using maximum.

The model consists of two linear segments joined at a hinge point:

\[f(x) = \max(y_1, y_2)\]

where:

  • \(y_1 = m_1 (x - x_1)\)

  • \(y_2 = m_2 (x - x_2)\)

Both lines pass through the hinge point \((h, y_h)\) where \(y_h = m_1 (h - x_1)\). The second slope is constrained by:

\[m_2 = m_1 \frac{h - x_1}{h - x_2}\]

This is the same as HingeMin but uses np.maximum instead of np.minimum, suitable for V-shaped patterns opening upward.

m1

Type:

Type:

>>> fit = HingeMax(x, y, guess_h=5.0)
>>> fit.make_fit()
>>> print(f"Hinge at x={fit.popt['h']:.2f}")

class HingeAtPoint(xobs, yobs, guess_xh: float = 400.0, guess_yh: float = 0.5, **kwargs)[source]

Bases: FitFunction

Piecewise linear function passing through a specified hinge point.

The model consists of two linear segments that both pass through the hinge point \((x_h, y_h)\):

\[f(x) = \min(y_1, y_2)\]

where:

  • \(y_1 = m_1 (x - x_1)\) with \(x_1 = x_h - y_h / m_1\)

  • \(y_2 = m_2 (x - x_2)\) with \(x_2 = x_h - y_h / m_2\)
