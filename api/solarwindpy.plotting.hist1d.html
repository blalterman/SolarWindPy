Create a one-dimensional histogram.

Parameters:

  • x (pandas.Series) ‚Äì Data from which to create bins.

  • y (pandas.Series or None, optional) ‚Äì Values to aggregate in bins of x. If None, counts of x are used.

  • logx (bool, optional) ‚Äì If True, compute bins in logarithmic space.

  • axnorm ({"t", "d", None}, optional) ‚Äì Normalisation applied to the histogram. "t" uses total counts and "d" yields a density.

  • clip_data (bool, optional) ‚Äì Remove extreme values at the 0.001 and 0.999 percentiles before binning or aggregation.

  • nbins (int or array-like, optional) ‚Äì Binning strategy passed to numpy.histogram_bin_edges() or pandas.cut() depending on the input type.

  • bin_precision (int, optional) ‚Äì Precision for decimal bin edges.

  • fcn (None, str, aggregative function, or 2-tuple) ‚Äì Passed directly to {self.__class__.__name__}.agg. If None, use the default aggregation function. If str or a single aggregative function, use it. If a 2-tuple is passed, then the first element aggregates and the second element calculates an uncertainty.

  • transpose_axes (bool) ‚Äì If True, plot independent values on y-axis and dependent values on x-axis. Primary use case is plotting 1D projection of 2D plot adjascent to 2D axis.

  • gaussian_filter_std (int) ‚Äì If > 0, apply scipy.ndimage.gaussian_filter to the z-values using the standard deviation specified by gaussian_filter_std.

  • gaussian_filter_kwargs (None, dict) ‚Äì If not None and gaussian_filter_std > 0, passed to scipy.ndimage.gaussian_filter()

  • plot_window (bool) ‚Äì Requires two functions passed to fcn. Instead of error bars, plots the uncertainty window as a semi-transparent band.

  • plot_window_edges (bool) ‚Äì If True, plot solid lines at the window boundaries.

  • kwargs ‚Äì Passed directly to ax.plot.
