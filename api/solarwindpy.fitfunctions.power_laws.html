solarwindpy.fitfunctions.power_laws module

Utilities for fitting power-law models.

This module provides FitFunction subclasses to fit power-law relations of the form f(x) = A x^b with optional offsets or centering. These classes supply sensible initial guesses and convenience properties for plotting and LaTeX reporting.
