solarwindpy.plotting.tools moduleÔÉÅ

Utility functions for common matplotlib tasks.

These helpers provide shortcuts for creating figures, saving output, and building grids of axes with shared colorbars.

Create a grid of subplots with a scaled figure size.

Parameters:

  • nrows (int, optional) ‚Äì Number of subplot rows.

  • ncols (int, optional) ‚Äì Number of subplot columns.

  • scale_width (float, optional) ‚Äì Factor applied to the default figure width.

  • scale_height (float, optional) ‚Äì Factor applied to the default figure height.

  • **kwargs ‚Äì Additional keyword arguments passed directly to matplotlib.pyplot.subplots().

Returns:

  • fig (matplotlib.figure.Figure)

save(fig, spath, add_info=True, info_x=0, info_y=0, log=True, pdf=True, png=True, **kwargs)[source]

Save a figure in both PDF and PNG formats.

Parameters:

  • fig (matplotlib.figure.Figure or matplotlib.axes.Axes) ‚Äì The figure or axis to save.

  • spath (pathlib.Path) ‚Äì Base path for the output files. The appropriate extension will be added automatically.

  • add_info (bool, optional) ‚Äì If True, add an attribution and timestamp to the bottom left of the PNG version.

  • info_x (float, optional) ‚Äì X-position of the attribution text in figure coordinates.

  • info_y (float, optional) ‚Äì Y-position of the attribution text in figure coordinates.

  • log (bool, optional) ‚Äì If True, write information about the saved files to alog.

  • pdf (bool, optional) ‚Äì Save a PDF version of the figure.

  • png (bool, optional) ‚Äì Save a PNG version of the figure.

  • **kwargs ‚Äì Additional keyword arguments passed to Figure.savefig().

Return type:

None

Examples

>>> fig, ax = subplots()
>>> save(fig, Path('my_plot'))

multipanel_figure_shared_cbar(nrows: int, ncols: int, vertical_cbar: bool = True, sharex: bool = True, sharey: bool = True, **kwargs)[source]

Create a grid of axes that share a single colorbar.

This is a lightweight wrapper around build_ax_array_with_common_colorbar() for backward compatibility.

Parameters:

  • nrows (int) ‚Äì Shape of the axes grid.

  • ncols (int) ‚Äì Shape of the axes grid.

  • vertical_cbar (bool, optional) ‚Äì If True the colorbar is placed to the right of the axes; otherwise it is placed above them.

  • sharex (bool, optional) ‚Äì If True share the respective axis limits across all panels.

  • sharey (bool, optional) ‚Äì If True share the respective axis limits across all panels.

  • **kwargs ‚Äì Additional arguments controlling layout such as figsize or grid ratios.

Returns:

  • fig (matplotlib.figure.Figure)

  • axes (ndarray of matplotlib.axes.Axes)

  • cax (matplotlib.axes.Axes)

Examples

>>> fig, axs, cax = multipanel_figure_shared_cbar(2, 2)
