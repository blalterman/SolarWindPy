solarwindpy.instabilities.verscharen2016 module

Instability thresholds from Verscharen et al. (2016).

The empirical fits of [] are implemented to evaluate when a plasma becomes unstable in the (beta, R) plane.

References

solarwindpy.instabilities.verscharen2016.beta_ani_inst(beta, a=None, b=None, c=None)[source]

Return the anisotropy threshold for a given beta.

Implements Eq. (5) of []:

\[R_p = 1 + \frac{a}{(\beta_{\parallel,p} - c)^b}\]

where $p$ is defined assuming only a single proton population is fit.

Parameters:

  • beta (array-like) – Parallel proton beta.

  • a (float) – Fit parameters from Verscharen et al. (2016).

  • b (float) – Fit parameters from Verscharen et al. (2016).

  • c (float) – Fit parameters from Verscharen et al. (2016).

Returns:

Threshold anisotropy values.

Return type:

numpy.ndarray

Examples

>>> beta = np.logspace(-2, 2, 5)
>>> beta_ani_inst(beta, a=0.367, b=-0.408, c=0.011)

Module contents

Tools for evaluating plasma instabilities.

This subpackage bundles functionality for visualizing and quantifying kinetic instabilities in the solar wind. It exposes the following modules:

beta_ani

Convenience wrappers for creating anisotropy plots.

verscharen2016

Thresholds and utilities based on the fits presented in [].
